🧪 BOT API Tester

ผ่าน Cloudflare Worker proxy — แก้ปัญหา CORS
⚠️ ทำไมต้องผ่าน Worker?
Browser บล็อก JavaScript ไม่ให้เรียก API ข้าม domain โดยตรง (CORS policy)
GitHub Pages → BOT API = ❌ โดนบล็อกทันที
GitHub Pages → Cloudflare Worker → BOT API = ✅ ทำงานได้
⚙️ Step 1 — ตั้งค่า Cloudflare Worker
ต้องทำใน Cloudflare ก่อน:
1. Workers → iaa-ingest → Settings → Variables and Secrets
2. เพิ่ม Secret: BOT_API_KEY = token ใหม่ที่ได้จาก portal.api.bot.or.th
3. paste ingest_worker_phase5.js → Save & Deploy
(Worker มี /bot/status, /bot/series, /bot/observations แล้ว)
INGEST_SECRET_TOKEN (ใช้ auth กับ Worker)
กด "ทดสอบ Worker" ก่อนครับ
📡 Step 2 — ทดสอบ BOT API ผ่าน Worker
```
1. ดู Categories ทั้งหมด
2. Series ตาม Category
3. Observations (ข้อมูลจริง)
4. Quick Test — Series รู้จัก
BOT API Status (ผ่าน Worker)
```

รอใส่ Worker Token แล้วกด ทดสอบ Worker...
