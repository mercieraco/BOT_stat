🚀 ขั้นตอน connect จริง
1
ขอ BOT API Key
apiportal.bot.or.th → สมัคร Account → Subscribe Statistics API → ได้ Client ID
2
รัน D1 Schema
Cloudflare → D1 → iaa-consensus-db → Console → paste schema_bot.sql
3
เพิ่ม Secret ใน Worker
CF Workers → iaa-ingest → Settings → Variables → BOT_API_KEY = Client ID
4
Deploy Worker routes
paste bot_worker_routes.js ใน ingest-worker.js → Save & Deploy
5
เลือก Mode ด้านบน
เลือก CF Worker → ใส่ INGEST_SECRET_TOKEN → กดตรวจสอบ Status
